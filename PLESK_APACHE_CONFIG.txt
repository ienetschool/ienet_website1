APACHE CONFIGURATION FOR IENET.ONLINE IN PLESK

=== IMMEDIATE FIX ===

In your Plesk Apache & nginx Settings (as shown in screenshot):

1. ADDITIONAL APACHE DIRECTIVES FOR HTTP:
Add this to the "Additional directives for HTTP" box:

DocumentRoot /var/www/vhosts/vivaindia.com/ienet.online/public
DirectoryIndex index.html

<Directory "/var/www/vhosts/vivaindia.com/ienet.online/public">
    Options -Indexes +FollowSymLinks
    AllowOverride All
    Require all granted
    
    # Enable rewrite engine
    RewriteEngine On
    RewriteBase /
    
    # Serve existing files directly
    RewriteCond %{REQUEST_FILENAME} -f
    RewriteRule ^.*$ - [L]
    
    # Serve existing directories directly  
    RewriteCond %{REQUEST_FILENAME} -d
    RewriteRule ^.*$ - [L]
    
    # Fallback to index.html for all other requests
    RewriteRule ^.*$ /index.html [L]
</Directory>

2. ADDITIONAL APACHE DIRECTIVES FOR HTTPS:
Add this to the "Additional directives for HTTPS" box:

DocumentRoot /var/www/vhosts/vivaindia.com/ienet.online/public
DirectoryIndex index.html

<Directory "/var/www/vhosts/vivaindia.com/ienet.online/public">
    Options -Indexes +FollowSymLinks
    AllowOverride All
    Require all granted
    
    RewriteEngine On
    RewriteBase /
    
    RewriteCond %{REQUEST_FILENAME} -f
    RewriteRule ^.*$ - [L]
    
    RewriteCond %{REQUEST_FILENAME} -d
    RewriteRule ^.*$ - [L]
    
    RewriteRule ^.*$ /index.html [L]
</Directory>

3. NGINX SETTINGS:
In the nginx settings section, add:

location / {
    root /var/www/vhosts/vivaindia.com/ienet.online/public;
    try_files $uri $uri/ /index.html;
    index index.html;
}
