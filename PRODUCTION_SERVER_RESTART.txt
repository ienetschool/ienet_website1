# PRODUCTION SERVER RESTART COMMANDS

## If using Plesk Control Panel:
1. Go to "Node.js" section
2. Click "Restart App" 
3. Wait for restart confirmation

## If using SSH/Command Line:
```bash
# Find and kill existing Node.js processes
ps aux | grep node
kill [process_id]

# Or kill all node processes
pkill -f node

# Start your updated server
cd /path/to/your/website
node updated-production-server-with-fixes.cjs
```

## If using PM2:
```bash
pm2 restart all
# Or specific app
pm2 restart your-app-name
```

## If using hosting panel:
1. Look for "Node.js Apps" or "Application Manager"
2. Click "Restart" button
3. Verify it shows "Running" status
