HOW TO UPLOAD THE INDEX.HTML FILE TO YOUR SERVER

=== METHOD 1: Via SSH (Recommended) ===

1. SSH into your server:
   ssh root@5.181.218.15

2. Create the directory and file:
   cd /var/www/vhosts/vivaindia.com/ienet.online
   mkdir -p public

3. Create the index.html file:
   nano public/index.html

4. Copy and paste the entire contents of the index.html file from this project

5. Save and exit:
   Ctrl+X, then Y, then Enter

6. Set permissions:
   chown -R ienet:ienet public/
   chmod -R 755 public/

=== METHOD 2: Via Plesk File Manager ===

1. Login to Plesk
2. Go to Files > File Manager
3. Navigate to: /var/www/vhosts/vivaindia.com/ienet.online/
4. Create new folder called "public"
5. Enter the public folder
6. Upload the index.html file OR create new file called "index.html"
7. If creating new file, copy and paste the HTML content
8. Set permissions: 755 for directories, 644 for files

=== METHOD 3: Via FTP/SFTP ===

1. Connect via FTP client (FileZilla, WinSCP, etc.)
2. Server: 5.181.218.15
3. Username: root (or your FTP user)
4. Navigate to: /var/www/vhosts/vivaindia.com/ienet.online/
5. Create "public" folder
6. Upload index.html to the public folder

=== VERIFY THE UPLOAD ===

After uploading, test via SSH:
ls -la /var/www/vhosts/vivaindia.com/ienet.online/public/
# Should show index.html file

curl -I http://localhost/
# Should return 200 OK
